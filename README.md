# GrabATicket

GrabATicket is a full-stack movie ticket booking web application that allows users to browse movies, view show timings, select seats, and book tickets through an intuitive interface.

## Features

- User authentication (Sign Up & Login)
- Browse and search movies
- View movie details and show timings
- Seat selection and ticket booking
- Booking history
- Wishlist
- Responsive user interface

## Tech Stack

### Frontend
- React
- Vite
- Tailwind CSS

### Backend
- Node.js
- Express.js
- TypeScript

### Database
- JSON Database (`db.json`)
